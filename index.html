快速开始

推荐设置：运行引导向导

系统要求

  • 运行时：Node ≥22
  • 支持：macOS、Linux、Windows（通过WSL2）
  • 包管理器：npm、pnpm 或 bun

推荐模型

强烈推荐使用 Anthropic Pro/Max (100/200) + Opus 4.5，具有长上下文优势和更好的提示注入防护。

安装命令 
# 安装
npm install -g moltbot@latest
# 或: pnpm add -g moltbot@latest

# 运行引导向导
moltbot onboard --install-daemon

# 启动Gateway
moltbot gateway --port 18789 --verbose

# 发送消息
moltbot message send --to +1234567890 --message "Hello from Moltbot"

# 与助手对话
moltbot agent --message "Ship checklist" --thinking high

安全模型

🔒 默认安全

工具在主会话的主机上运行，因此当只有您使用时，代理具有完全访问权限。

🛡️ 群组/频道安全

设置 sandbox.mode: "non-main" 以在非主会话（群组/频道）中运行Docker沙箱。

✅ DM配对

未知发送者会收到配对码，机器人不会处理他们的消息，直到被批准。

聊天命令

在任何支持的渠道中发送这些命令

/status 紧凑的会话状态
/new 或 /reset 重置会话
/compact 压缩会话上下文
/think <level> 设置思考级别
/verbose on|off 切换详细模式
/usage off|tokens|full 每响应使用页脚
/restart 重启gateway（仅限所有者）
/activation mention|always 群组激活切换
